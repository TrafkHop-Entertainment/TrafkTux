# TrafkTux

An Arch Distro from Trafkhop Enjoyers for Trafkhop-Enjoyers!

## Introduction
TrafkTux is not a minimalistic Distrobution.

## Features

TrafkTux has a lot of features:

### Visually it has a custom:
* Bootscreen
* Grub Menu
* SDDM login screen
* Rofi app menu
* waybar
* background images
* hyprland config

### Feature wise it has:
* A GTK App for editing the rofi app menu and waybar shortcuts
* 5 Hyprland layouts: dwindle, Master, Scrolling, Floating, SingleWindow
* popup menus on the waybar somewhat like on kde or windows
* a hideall windows button for better background viewing experience
* a workspace viewer for viewing the 9 workspaces
* A Rofi app menu where you can create folders with app or more folders in them
* waybar shortcuts to your favourite apps
